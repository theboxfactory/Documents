Here are some examples of markdown syntax using the CommonMark standard:

1. Headers:

```
# Heading 1
## Heading 2
### Heading 3
```

2. Emphasis:

```
*Italic text*
**Bold text**
```

3. Lists:

```
- Item 1
- Item 2
  - Subitem 1
  - Subitem 2
1. Ordered item 1
2. Ordered item 2
```

4. Links:

```
[Link text](https://www.example.com)
```

5. Images:

```
![Alt text](image.jpg)
```

6. Code blocks:

````
```python
def hello_world():
   print("Hello, World!")
````

```
7. Blockquotes:
```

> This is a blockquote.

```
8. Horizontal rule:
```

---

```
9. Tables:
```

| Column 1 | Column 2 | | -------- | -------- | | Cell 1 | Cell 2 |

```
These are just a few examples of markdown syntax. You can find more information in the CommonMark specification.
```

10. Code Blocks

```
function HelloWorld() {
  return 'Hello World';
}
```
